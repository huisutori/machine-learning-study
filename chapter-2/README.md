# 2. 데이터 다루기

**2.1 훈련 세트와 테스트 세트**

+ 지도학습과 비지도 학습의 차이를 배운다.
+ 모델을 훈련 시키는 훈련 세트와, 모델을 평가하기 위한 테스트 세트로 데이터를 나누어 학습한다.

**2.2 데이터 전처리**

+ 올바른 결과 도출을 위하여, 데이터를 사용하기 전에 데이터 전처리 과정을 거친다.
+ 표준 점수로 특성의 스케일을 변환하는 방법을 배운다.
